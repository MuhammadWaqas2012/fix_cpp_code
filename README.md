# fix_cpp_code
A cursed collection of the worst C++ code imaginable. Built for developers who want to practice debugging, refactoring, and surviving bad code in the real world.

## 🔧 Your Mission
Fix the code. Improve it. Make it human.

## 💀 What You'll Find
- Horrible logic
- Useless loops
- Abusive macros
- Random chaos

## 🚀 Goal
Turn cursed code into clean, efficient, production-ready C++.
